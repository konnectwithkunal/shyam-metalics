# shyam-rebuild

Vite + React + Tailwind CSS + Framer Motion starter scaffold tailored to recreate the ShyamMetallics homepage.

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Start dev server:
   ```
   npm run dev
   ```

3. Build:
   ```
   npm run build
   ```

## Notes

- Replace placeholder images in `public/` (e.g. `hero-bg.jpg`) with actual images.
- The project includes a simple `useInView` hook (IntersectionObserver) and Framer Motion animations.
- Tailwind is pre-configured. Modify `tailwind.config.js` as needed.
